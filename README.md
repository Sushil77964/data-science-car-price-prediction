# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the selling price of a used car using Machine Learning algorithms. Multiple regression models were trained and compared to achieve the best performance.

---

## 🚀 Features

- Data Cleaning
- Exploratory Data Analysis (EDA)
- Data Visualization
- Feature Engineering
- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor
- Streamlit Web Application

---

## 🛠 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit
- Jupyter Notebook

---

## 📊 Dataset

- 301 Records
- 9 Original Features

Target Variable:
- Selling Price

---

## 🤖 Models Used

| Model | R² Score |
|--------|---------:|
| Linear Regression | 0.8467 |
| Decision Tree | 0.9544 |
| Random Forest | **0.9682** |
